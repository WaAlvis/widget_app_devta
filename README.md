# widget_app_devta  
A Flutter application that shows different components of this framework.

State Manager with [Riverpod](https://riverpod.dev/)

* Cards
* Buttons
* Progress
* Snackbars
* Dialogs
* Animated Container
* Checkbox
* ListTiles
* Switchs
* Tutorial page
* Infinite Scroll with Api

* Light and dark theme with different Seed colors
